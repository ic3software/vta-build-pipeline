VTA Auth Demo

A static-file harness for exercising the VTA's auth surface from a browser.

1. Connect

Must be reachable from this page. The VTA must include this page's origin in [server] cors_origins — see the README.

2. Bootstrap auth (paste JWT)

First-time setup: get a JWT from pnm auth show-token, paste it here. Unlocks Step 3 (enrol passkey) and Step 5 (sessions + trust-tasks) without going through passkey login.

3. Enrol a passkey VM

Adds a WebAuthn passkey as a verificationMethod on a WebVH-managed DID. Requires an admin JWT (from Step 2). The browser drives navigator.credentials.create; the VTA publishes the new VM via a WebVH LogEntry.


  


  

    
4. Passkey login

    
Once a DID has a passkey VM (from Step 3 or already enrolled), use it to obtain a JWT directly — no DIDComm round-trip.
