What lives here

This is the default landing page. The operator hasn't configured a public website yet (website.root_dir is unset).

  • Operators point /admin/ at the admin UX to manage the community.
  • Members apply via the join flow on the API at POST /v1/join-requests.
  • Developers hit the JSON API under /v1/*.

Replace this page by setting website.root_dir in the daemon config and populating the directory, or by uploading a bundle via POST /v1/website/deploy.
